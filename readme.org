*** Limitations

- Tables are expected to begin and end with pipes.

Good:

#+BEGIN_SRC markdown
| Colour | Favourite  |                           | Colour   | Favourite                    |
|--------|----------------------------|           |----------|------------------------------|
| Red              | yes        |            =>  | Red      | yes                          |
| Blue   | not really                   |         | Blue     | not really                   |

#+END_SRC

Bad:

#+BEGIN_SRC markdown
 Colour | Favourite
--------|----------------------------
 Red              | yes                       =>   Will not properly format.
 Blue   | not really
#+END_SRC
